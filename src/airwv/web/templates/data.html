Where the data comes from

Everything on this site is built from public government and open data — and nearly all of it needs no API key. This page catalogs every layer, its source, and how fresh it is. Half the point of AirWV is showing that this data is out there and connectable — so if you want to build your own tool, start with our Data Sources field guide (the how-to for tapping each source) and the scripts/ that pull them.
